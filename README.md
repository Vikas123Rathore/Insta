# 📸 Instagram Clone Backend

This is the backend API for an Instagram-like social media application built using MERN stack.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 📸 Create, Like, Comment on Posts
- 💾 Save / Unsave Posts
- 📖 Stories (auto delete after 24 hours)
- 👀 Story Viewers (seen system)
- 👤 User Profile System

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Cloudinary (for media upload)
- JWT (Authentication)
